数据收集

许可证程序已经做好了，接下来是爬取数据，书中介绍了几种数据内容，安装处理数据的方式分为：文本内容、时间序列、关系和地理。但也可以理解为：静态数据和动态数据。静态数据主要包括：本人关注信息流、指定用户发布信息、指定用户粉丝和关注信息；动态数据主要是指定标签和内容的信息流（这里就涉及到不同用户的发布信息）

静态数据 REST API
